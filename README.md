# Xander Makes Games — Full Portfolio Site

A dark, cinematic game programming portfolio built for GitHub Pages.

## Pages included

- `index.html` — cinematic home page
- `projects.html` — project hub
- `projects/cams-vr.html` — CAMS VR case study
- `projects/proof-pending.html` — Proof Pending case study
- `projects/adventure-game.html` — 2D Adventure Game case study
- `projects/racing-game.html` — Racing Game case study
- `projects/rat-heist.html` — Rat Heist / Rat Survival concept page
- `about.html` — about page
- `skills.html` — skills page
- `devlog.html` — development journal
- `styles.css` — full dark theme
- `script.js` — mobile nav and scroll animation
- `CNAME` — custom domain file for GitHub Pages

## Important

Keep the `CNAME` file in the root of your repo. It contains:

```txt
xandermakesgames.com
```

## Add your profile photo

Add your photo at:

```txt
images/profile.jpg
```

## Update links

Search for:

```txt
href="#"
```

Replace those with your real GitHub, LinkedIn, Itch.io, email, demo, or video links.

## Deploy

Replace your repo files with these files, commit, and push. GitHub Actions should redeploy the site automatically.
