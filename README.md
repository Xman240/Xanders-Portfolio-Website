# Xander Puhlmann Game Programming Portfolio

This is a custom static portfolio website built from the supplied Word document.

## Files

- `index.html` — home page
- `projects.html` — project showcase
- `about.html` — about, education, and skills
- `styles.css` — full website styling
- `script.js` — mobile menu and footer year
- `images/` — extracted project images from the Word document

## Add your profile photo

Add a personal photo here:

```txt
images/profile.jpg
```

The homepage is already set up to display it.

## Publish with GitHub Pages

1. Replace your repo files with these files.
2. Commit and push to GitHub.
3. Go to **Settings → Pages**.
4. Select the `main` branch and `/root`.
5. Wait for GitHub Pages to deploy.
6. Hard refresh your browser with `Ctrl + F5`.

## Editing project links

Search for `href="#"` in the HTML files and replace those placeholders with GitHub, itch.io, demo, or video links when ready.
