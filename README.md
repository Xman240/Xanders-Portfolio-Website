# Xander Makes Games — Dark Portfolio Upgrade

This package is a darker, more polished upgrade of the portfolio. It adds:

- Dark cinematic colour theme
- Dev Log page
- Updated navigation
- Project breakdown cards
- Scroll reveal animations
- CNAME file for xandermakesgames.com

## Upload
Replace the repo files with this package, commit, and push. GitHub Pages should redeploy automatically.

## Profile photo
Add a file named `profile.jpg` inside the `images` folder.

## Links
Search for `href="#"` and replace placeholders with your GitHub, LinkedIn, Itch.io, demo, or email links.
