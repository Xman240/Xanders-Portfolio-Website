Overview

I learned how to make 2D games using tilemaps and animated sprites. I also built a save system and used prefab-based map navigation so certain doors could lead to new maps.

Technical Focus

Unity tilemaps, animated sprites, prefab navigation, map transitions, save system logic, and enemy/player animation.

Challenge

The challenge was connecting several small systems together into something that behaved like a complete game structure.

What I Learned

I improved my understanding of 2D workflows, saved state, scene/map transitions, and organized prefab-based systems.

Next Step

Add gameplay footage, clean up the UI, and write a stronger breakdown of the save system.
