Proof Pending

A multiplayer cryptid horror game concept where players search for evidence and survive the night.

← Back to Projects Demo / Repo Link

Overview

Proof Pending is my first full Game Design Documented game and the first project where I attempted to build a vertical slice. Most of the core systems are in place, though final assets are still needed.

Technical Focus

Unity, Netcode for GameObjects, multiplayer setup, survival mechanics, evidence gathering, and two-phase game loop design.

Challenge

The biggest challenge was self-teaching multiplayer code while also managing scope, gameplay flow, and technical reliability.

What I Learned

I learned that multiplayer architecture needs planning early, and that features need to be broken into small testable networked behaviours.

Next Step

Polish the two-phase loop, improve asset quality, continue multiplayer reliability testing, and add a short gameplay trailer.
