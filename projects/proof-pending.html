Proof Pending

A multiplayer cryptid horror vertical slice focused on evidence gathering and survival.

← Back to Work

Overview

Proof Pending is my first full Game Design Documented game and the first project where I attempted to create a vertical slice. The concept is a multiplayer horror game where players search a forest for cryptid evidence and survive the night.

My Role

I self-taught multiplayer setup in Unity using Netcode for GameObjects and built core systems for the prototype.

Challenge

The biggest challenge was learning multiplayer code while also designing a complete two-phase gameplay loop.

What I Learned

I learned that multiplayer systems need careful planning, small testable features, and constant debugging.

Next Step

Polish assets, strengthen the game loop, and add gameplay footage.
