Racing Game

A one-lap racing game focused on save data, profiles, best times, ghost car playback, and leaderboards.

← Back to Projects Demo / Repo Link

Overview

This project is a simple racing game where the player completes one lap around the track. The game saves the best time and plays it back as a ghost car that the player competes against.

Technical Focus

Profile creation, car type and colour selection, data saving, best time tracking, leaderboard storage, and ghost replay.

Challenge

The main challenge was saving and managing player/profile data cleanly across sessions.

What I Learned

I learned how much I enjoy creating data-driven features like profiles, saved best times, and leaderboards.
