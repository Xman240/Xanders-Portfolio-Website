CAMS VR Medical Simulation

A NAIT capstone proof of concept for immersive VR medical simulation.

← Back to Projects

Overview

This project was built for the Center of Advanced Medical Simulation at NAIT. The goal was to create a realistic child using Unreal's Metahuman and an AI Chatbot to engage in immersive and dynamic medical scenarios. The child character was questioned by players so that nursing students could practice getting pediatric medical history.

My Role

My role in this project was to set up the AI Chatbot and handle all the connections to the metahuman including character creation and custom dialogue triggered animations.

Challenge

The main challenge was integrating the AI Chatbot into the Metahuman, since the ai conversation system was still in beta development. I got great experience teaching myself using dev logs and forums.

What I Learned

I learned so much from this project. This was my first experience using Unreal Engine as well as coding in VR. This was also my first time working for a client. I was able to develop my problem-solving skills and gain valuable experience in client communication.
